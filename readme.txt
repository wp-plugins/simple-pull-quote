== Description ==

The Simple Pull Quote Wordpress Plugin provides an easy way for you to insert pull quotes into your posts and pages. 


== Installation ==

1. Upload the entire contents of the `simple-pull-quote.zip` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Usage =

1. Create a custom field name of "quote" with a value of the text to be quoted
2. Type "[quote]" in your post or page wherever you want the pull quote to appear.
<br /><br />For more help on usage, visit the [Simple Pull Quote Homepage](http://themightymo.com/simple-pull-quote "Simple Pull Quote Wordpress Plugin") for a visual guide.

= How do I update the look of the pull quotes? =

To change the look of your pull quotes, simply edit the CSS file included in the plugin installation folders at "wp-content/plugins/simple-pull-quote/css/simple-pull-quote.css".  Style the pull quotes with the CSS class of “simplePullQuote”.
